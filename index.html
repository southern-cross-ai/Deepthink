Local AI Dev Platform

Run AI models locally. Build AI apps fast. No cloud needed.

🚀 What We’re Building

We’re creating a Docker-based platform that bundles ollama (model runner), langchain (app logic), and gradio (UI) into a ready-to-use container. Just pull the image and build your AI app.

🔧 Key Features

  • Run LLMs locally via ollama
  • Use langchain for modular AI workflows
  • Launch an interactive gradio UI right from the container
  • Designed for fast prototyping and offline usage

📦 What You Get

  • A clean Docker image with all components pre-installed
  • Template code for building your own AI agent
  • Docs & usage guides to get started in minutes
  • Optional: RAG module, system prompt config, multi-turn memory
